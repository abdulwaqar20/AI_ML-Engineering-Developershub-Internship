# Task 4: General Health Query Chatbot

## Objective
The objective of this task is to develop a chatbot capable of answering general
health-related questions using a Large Language Model (LLM), while ensuring
safe, clear, and responsible responses.

---

## Tools & Technologies
- **Programming Language:** Python  
- **LLM Provider:** Groq API  
- **Model Used:** LLaMA 3.1 8B Instant  
- **Development Environments:** VS Code, Google Colab  

---

## Features
- Interactive command-line chatbot
- Prompt engineering for friendly and understandable responses
- Safety filtering to prevent harmful medical advice
- Avoids diagnosis, prescription, and treatment recommendations
- Supports general health education use cases

---

## Example Queries
- What causes a sore throat?
- Is paracetamol safe for children?
- How can I improve my sleep quality?

---

## Safety & Ethical Considerations
- The chatbot provides **general health information only**
- Medical diagnosis and prescriptions are intentionally restricted
- Users are advised to consult a healthcare professional when necessary

---

## Disclaimer

This chatbot is for educational and informational purposes only and is not
a substitute for professional medical advice, diagnosis, or treatment.
